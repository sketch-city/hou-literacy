# Houston Literacy Map
---

A map of Houston Adult Literacy, built for the Houston Center for Literacy.

Map is viewable at [sketch-city.github.io/hou-literacy](http://sketch-city.github.io/hou-literacy/)


## Data Sources
Primary Data from [COH GIS Open Data](http://cohgis.mycity.opendata.arcgis.com/)

* [Language Spoken at Home by Census Block Group (2010)](http://cohgis.mycity.opendata.arcgis.com/datasets/700dc2edd41e484da160c882739a981b_2/features/188)
* [Median Household Income by Census Block Group (2010)](http://cohgis.mycity.opendata.arcgis.com/datasets/7ffcdb0ec7d04f0bb47c74eee5ee275b_5?uiTab=table)
* [Educational Attainment by Census Block Group (2010)](http://cohgis.mycity.opendata.arcgis.com/datasets/700dc2edd41e484da160c882739a981b_0/features/130)
